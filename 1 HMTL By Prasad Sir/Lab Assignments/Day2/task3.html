Quotes and Blockquotes

These tags are used for quoting text from other sources, distinguishing between short inline quotes and longer, block-level quotes.

Short Quotes

The q tag is used for short, inline quotations. My friend said, “I'll be there in five minutes!” before hanging up.

Another common phrase is, “Actions speak louder than words.”

Block Quotes

The blockquote tag is used for longer quotations that are typically displayed as a separate block of text. For example:

“Two roads diverged in a yellow wood, And sorry I could not travel both And be one traveler, long I stood And looked down one as far as I could To where it bent in the undergrowth.”

— Robert Frost, The Road Not Taken

This blockquote demonstrates how a large section of quoted text is rendered, typically with indentation, and can include a citation using the cite tag.

Semantic Text Elements (Beyond Basic Styling)

These tags add specific semantic meaning to text, which can be useful for accessibility, search engines, and more precise data representation.

Abbreviation and Acronyms

The abbr tag is the standard HTML way used to define both “abbreviations” and “acronyms.” It ' s important for providing the full, expanded form of a shortened word or phrase, which helps with accessibility and search engine optimization.

You use the title attribute to provide the full description. When a user hovers over the abbreviated text, the full form usually appears as a tooltip.
