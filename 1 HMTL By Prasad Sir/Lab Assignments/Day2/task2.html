Mathematical expressions or ordinal indicators use superscripts, such as E=mc² or 10ᵗʰ, where the '2' and 'th' are sup elements.

Marked (Highlighted) Text

The mark tag is used to highlight or mark text that is of particular relevance in the context of the surrounding content, often rendered with a yellow background.

Small Text

This is normal text, and this is small text using the small tag. It typically represents side comments, copyright, or legal disclaimers.

Cited Work

The cite tag is used to denote the title of a work. For example, as famously quoted from Hamlet by William Shakespeare, "To be or not to be, that is the question."

Another example: The theory was first proposed in On the Origin of Species.

Code, Variables, and Preformatted Text

These tags are crucial for displaying programming code, user input, sample output, and preserving text formatting.

Code Snippets

For inline code, use the code tag. For example, to print "Hello, World!" in Python, you would use print("Hello, World!").

When defining a function in JavaScript: function greet() { console.log("Hi!"); }

Keyboard Input

The kbd tag indicates user input, typically from a keyboard. To save a document, press Ctrl + S.

Another example: To copy text, use Ctrl + C.

Sample Output

The samp tag represents sample output from a computer program. If the command fails, you might see: Error: Permission denied.

After a successful compilation, the console might display: Build successful (0 warnings, 0 errors).

Variables

The var tag is used to define a variable in a mathematical expression or programming context. In the formula for the area of a circle, A = πr², A represents the area and r represents the radius.

Another example:Let x be the number of apples.
