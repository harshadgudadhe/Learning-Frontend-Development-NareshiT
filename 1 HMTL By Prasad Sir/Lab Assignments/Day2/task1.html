HTML Presentational & Formatting Tags

This page provide an extensive demonstration of various HTML tags specifically designed for text formatting and presentation. It highlights how these tags influence the visual appearance and semantic meaning of content without relying on external CSS.

Text Styling and Emphasis Tags

These tags are used to apply common text styles like bold, italic, underline, and to convey different levels of emphasis or semantic meaning.

Bold and Strong Emphasis

This sentence contains text that is rendered in a bold font style using the b tag. The b tag is typically used for visually bolding text without adding any special semantic importance.

Conversely, this sentence has text that conveys strong importance using the strong tag. While it often renders as bold, its primary purpose is to indicate that the content is of great importance or seriousness.

Italic and Emphasized Text

Here, we have text that is displayed in an italic font using the i tag. The i tag is generally used for text that needs to stand out visually, such as technical terms, foreign words, or thoughts.

This sentence showcases text that is emphasized using the em tag. The em tag adds semantic emphasis, implying that the content should be stressed or pronounced differently.

For instance, use em when you want to emphasize a word in a sentence to change its meaning, and i tag for titles of works or scientific names.

Underlined and Inserted Text

This text is underlined using the u tag. Historically, this was used for visual underlining, but it's now often discouraged for non-hyperlink text to avoid confusion. It can be used to represent text that is unarticulated but explicitly rendered as if underlined, such as misspellings.

This text is inserted using the ins tag. This tag semantically indicates content that has been added to a document, often rendered with an underline.

You can also specify a datetime attribute for ins to indicate when the content was inserted: New content added on Jan 15, 2023

Strikethrough and Deleted Text

This text is struck through using the s tag. This tag indicates content that is no longer accurate or relevant, but should not be removed from the document. It's purely presentational.

This text is deleted using the del tag. This tag semantically indicates content that has been removed from a document, often rendered with a strikethrough.

Similar to ins, del can also have a datetime attribute: Old price: $29.99 New price: $24.99
